Sobre este Trabalho

Aplicação para gerenciar listas de presentes: criar e compartilhar listas, cadastrar produtos e controlar quantidades. Permite buscar produtos por nome, adicionar itens às listas e acompanhar o que já foi comprado. Desenvolvida para facilitar a organização de eventos e presentes, com interface simples e armazenamento local dos dados.


A ideia do TP4 é apenas a implementação do CRUD de Produto na web utilizando o Local Storage API
